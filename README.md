# HTML Resume Page

This project is a single-page resume website built using only HTML, based on the provided reference layout.

The project focuses on writing clean and semantic HTML without using CSS or JavaScript.

## Features

The resume includes the following sections:

- Header / Introduction
- About Me
- Skills
- Experience
- Projects
- Education
- Contact Information
- Footer

Semantic HTML elements such as:
- `<header>`
- `<main>`
- `<section>`
- `<article>`
- `<address>`
- `<footer>`

have been used to improve structure and readability.

Dummy data has been used for the content.

## How to Run

1. Download or clone the repository
2. Open `index.html` in any web browser
